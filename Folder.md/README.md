# LONDON ROUTE TRANSFERS

Сайт транспортной компании с лендингом, формой бронирования и страницей менеджера на mock-данных.

## Стек

- **Next.js 15** (App Router)
- **React 19**, **TypeScript**
- **Tailwind CSS v4**
- **next-intl** (мультиязычность EN/RU)
- **react-hook-form** + **zod** (форма и валидация)
- **motion** (анимации)

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev
# Открыть http://localhost:3000

# Сборка для production
npm run build

# Старт production-сервера
npm run start

# Линтинг
npm run lint
```

## Страницы

| Маршрут | Описание | Рендеринг |
|---|---|---|
| `/` | Главная лендинговая страница | Server |
| `/booking` | Форма бронирования | Server + Client (форма) |
| `/manager` | Таблица бронирований (mock) | Server + Client (таблица) |
| `/privacy` | Политика конфиденциальности | Server |
| `/cookie` | Политика cookies | Server |
| `/terms` | Условия использования | Server |
| `/transport-terms` | Условия транспортных услуг | Server |

## Языки

Переключение языка — в шапке сайта (кнопки EN/RU).

## Форма бронирования

Поля формы:
- Имя и фамилия (обязательно)
- Компания (опционально)
- Телефон (обязательно)
- Email (обязательно)
- Тип услуги (обязательно)
- Адрес подачи (обязательно)
- Пункт назначения (обязательно)
- Дата поездки (обязательно)
- Время подачи (обязательно)
- Количество пассажиров (обязательно, >0)
- Meet & Greet (да/нет)
- Комментарий (опционально)
- Согласие на обработку данных (обязательно)

После отправки форма показывает success-состояние. Данные не сохраняются (будет на этапе 3).

## Mock-данные

На странице `/manager` 12 предзаполненных бронирований с возможностью:
- Поиска по имени, телефону, маршруту, ID
- Фильтрации по статусу (confirmed, pending, completed, cancelled)
- Просмотра статистики
